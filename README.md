# Python & AI Development Tools - B10 🚀

Repositori ini berisi serangkaian tugas praktikum yang berfokus pada penguasaan bahasa pemrograman Python, mulai dari logika dasar hingga manipulasi data untuk kebutuhan pengembangan *Artificial Intelligence*.

## 📌 Kesimpulan & Refleksi Pembelajaran

Sebagai seseorang yang mempelajari infrastruktur sistem dan AI, ketiga tugas ini memberikan transisi logis dari pemahaman memori dasar menuju pemrosesan data berskala industri.

### Tugas 4: Struktur Data (Collections & Comprehensions)
Pada modul ini, fokus utamanya adalah bagaimana sistem mengalokasikan dan mengatur data di dalam memori.
* **List, Tuple, Set, dan Dictionary** bukan sekadar variasi variabel, melainkan alat strategis. Misalnya, penggunaan `Set` terbukti sangat efisien untuk memfilter duplikasi data (seperti membersihkan *dataset*), sementara `Dictionary` memberikan struktur *key-value* yang fundamental bagi format JSON dan interaksi API.
* Penggunaan **Comprehensions** menyederhanakan *looping* menjadi satu baris operasi matematis yang lebih cepat dieksekusi oleh mesin.

### Tugas 5: Arsitektur Kode (Functions & OOP)
Modul ini mengubah cara pandang dari "menulis skrip prosedural" menjadi "membangun arsitektur perangkat lunak".
* **Functions** (dengan *Type Hinting*) memastikan bahwa input dan output memiliki kontrak data yang jelas, mencegah *bug* sistemik.
* **Object-Oriented Programming (Class)** memungkinkan kita membungkus data (atribut) dan logika (method) ke dalam satu entitas mandiri. Konsep ini sangat vital ketika mendevelop sistem yang kompleks, seperti entitas pemain dalam mekanika *game* atau agen dalam simulasi AI.

### Tugas 6: Data Science Fundamentals (NumPy, Pandas & I/O)
Ini adalah jembatan menuju implementasi nyata di lapangan.
* **NumPy** digunakan untuk komputasi numerik tingkat rendah. Modul ini jauh lebih cepat daripada *list* bawaan Python karena dieksekusi di level C, sangat krusial untuk kalkulasi matriks dalam AI.
* **Pandas** menyederhanakan manipulasi data tabular menjadi *DataFrame*, membuat proses analisis data empiris (seperti menghitung batas kelulusan) menjadi sangat efisien.
* **File I/O** memastikan bahwa hasil analitik dari sistem dapat diekspor dan dibaca di luar *environment* Python (disimpan ke `.txt`).

Secara keseluruhan, rangkaian ini mengajarkan bahwa pemrograman bukan sekadar menghafal *syntax*, melainkan menggunakan kerangka berpikir logis untuk memanipulasi data dan memecahkan masalah.
